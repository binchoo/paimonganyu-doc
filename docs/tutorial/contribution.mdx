# 여행 비서 페이몬! 함께하기

<TOCInline toc={toc} />


**「여행 비서 페이몬!」**과 친구를 맺고 이용해주셔서 감사합니다. :tada: 

해당 챗봇은 개인(비사업자)이 운영하는 비영리 챗봇으로서 언제나 리소스가 부족한 개인 프로젝트입니다. 

<u>여러분의 참여</u>가 여행 비서 페이몬 프로젝트의 개선에 도움이 될 거예요!

> ![페이몬 이모지](resources\kakao_channel_icon_50_50.png)**「여행 비서 페이몬!」 함께 개선하기!**
>
> 챗봇을 써보니 유용했어? 아니면 부족하다고 느꼈어? 여행자 목소리가 페이몬의 직무 개선에 도움이 돼~ 
>
> 그러니 언제든, 여행자의 지혜를 페이몬에게 전수해 달라구!

## 코드 개선하기

![](resources\operation\paimonganyu_repo.png)

*PaimonGanyu 깃허브 저장소* 



**여행 비서 페이몬!**의 소스 코드는 전부 깃허브에 올려져 있습니다.​ :point_right: [PaimonGanyu (깃허브)](https://github.com/binchoo/paimonganyu)



챗봇의 1차 배포까지 급하게 달린 감이 없지 않아, 코드 베이스에 아픈 부분이 많습니다. :skull_and_crossbones:

풀 리퀘스트나 코드 리뷰를 언제나 환영할게요. 기여할 수 있는 부분이 있다면 편하게 제안해 주세요~

## 운영 지원하기

![](resources\operation\paimonganyu_aws_arch.png)

**「여행 비서 페이몬!」**의 모든 인프라는 [카카오](https://i.kakao.com/) 및 [아마존 웹 서비스](https://aws.amazon.com/ko/?nc2=h_lg)의 클라우드 상에 구축되어 있습니다.

카카오 챗봇의 비즈니스 기능을 이용하지 않으므로 챗봇에 대해선 과금이 없으나, 페이몬 비서를 보좌하는 뒷 단의 AWS 자원들은 꾸준하게 비용이 지출됩니다!
